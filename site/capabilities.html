VIZIER Capabilities

VIZIER is a real AI agent running on a local machine. These are its connected capabilities — shown here for illustration, never executed from this page.

Architecture note: All capabilities below run on the local VIZIER backend and are shown here for illustration only. The public site never connects to Gmail, Calendar, Telegram, Supabase, or any of these accounts. Every action that touches external services requires human approval before it fires.

Gmail

Read • Draft • Send on approval

The agent reads your inbox, drafts context-aware replies, and proposes them to you. Nothing sends without an explicit approval step from the human.

Gmail API → VIZIER Reader
VIZIER Drafter → Approval Gate
Human says GO → Send
Human-approved sends OAuth 2.0 Read + draft
This runs on the local VIZIER backend and is shown here for illustration — the public site never connects to these accounts.

Grounded Research

Europe PMC • Cited • Tiered • Abstains

Queries Europe PMC for primary literature, extracts abstracts, tiers each source (1–4 by evidence strength), and abstains when abstracts are empty. No hallucinated citations.

Europe PMC API
↓ Fetch abstracts
Tier classifier (1–4)
↓
Feynman note with sources

Approval Gate

Human-in-the-loop • Full audit trail

Every action that touches the external world (send email, create event, post message) is queued for your explicit approval. The agent writes an audit log of every action it proposes and every decision you make.

↓
PENDING queue
↓ Human reviews
